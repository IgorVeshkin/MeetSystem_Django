Данный сайт был написан Вешкиным Игорем Ильичем для демонстрации своих профессиональных навыков программирования

На сайте были реализованы следующие функции:

  • Система учетной записи пользователя;
  • Разделение пользователей по ролям (Admin, User) и предоставление различного функционала соответственно;

      Admin может:
    1. Admin может добавлять события мероприятий в свое расписание;
    2. Admin может создавать/удалять мероприятия;
    3. Admin может создавать/удалять/редактировать события мероприятий;
    4. Admin может блокировать и разблокировать пользователей на сайте (заблокированные пользователи не могут войти в учетную запись и им нельзя отправлять сообщения после блокировки);
    5. Admin может видить всех заблокированных пользователей и тех, кто пожелал скрыть профиль в общем поиске;
    6. Admin не может быть заблокирован в системе;

      User может:
    1. User может добавлять события мероприятий в свое расписание;
    2. User видить ограниченное количество пользователей в поиске;
    3. User взаимодействовать с ограниченным количество пользователей системы;

    • При этом при попытке пользователя с ролью User перейти на страницы отведенные под Admin пользователь будет перенаправлен на главную страницу;

  • Поиск пользователей;
  • Настройка информации профиля (в том числе можно сокрыть отображение акаунта в поске);
  • Просмотр расписаний других пользователей;
  • Real-Time Чат-система, позволяющая общаться с другими пользователями. Возможна отправка, редактирование, удаление сообщений. Все изменения отобразяться у собеседника мгновенно!;
